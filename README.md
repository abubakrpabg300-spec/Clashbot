# 🏰 Clash Farm Bot

Телеграм бот с героями, заданиями, реферальной системой и пополнением через CryptoBot.

## Функции
- 🏰 Герои с пассивным доходом
- 📋 Задания
- 👥 Реферальная система
- 💎 Пополнение через @CryptoBot (автоматически)
- 💸 Вывод в TON
- 🎁 Ежедневный бонус
- 🎫 Промокоды
- 🔧 Админ-панель

## Установка
1. `pip install -r requirements.txt`
2. `python bot.py`
